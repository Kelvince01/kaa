Kaa API

API Documentation

Welcome to the Kaa API. This RESTful API provides endpoints for property listings, user management, bookings, payments, and more.

For comprehensive documentation, visit API Documentation.

Available Endpoints

  • /api/v1/auth - Authentication endpoints
  • /api/v1/admin - Admin endpoints
  • /api/v1/applications - Application endpoints
  • /api/v1/audits - Audit endpoints
  • /api/v1/analytics - Analytics endpoints
  • /api/v1/documents - Document endpoints
  • /api/v1/files - File endpoints
  • /api/v1/properties - Property management
  • /api/v1/units - Property unit management
  • /api/v1/favourites - Favourite property management
  • /api/v1/references - Reference management
  • /api/v1/reviews - Property review endpoints
  • /api/v1/ai - AI endpoints
  • /api/v1/users - User management
  • /api/v1/monitoring - Monitoring endpoints
  • /api/v1/notifications - Notification endpoints
  • /api/v1/webhooks - Webhook endpoints
  • /api/v1/bookings - Booking functionality
  • /api/v1/payments - Payment processing
  • /api/v1/messages - Messaging system
  • /api/v1/contracts - Rental contract management
  • /api/v1/tenants - Tenant management
  • /api/v1/members - Member management

Health Check

Server health check is available at Health Check.

Detailed server health check is available at Detailed Health Check.

Monitoring

Server and system health and metrics are available at Metrics (JSON format).

Server health and metrics are available at Prometheus Metrics (Prometheus format).
